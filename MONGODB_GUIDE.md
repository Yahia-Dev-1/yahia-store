# كيفية إنشاء قاعدة بيانات MongoDB مجانية

عشان الموقع يشتغل من أي مكان ويحفظ البيانات أونلاين، لازم نعمل قاعدة بيانات على **MongoDB Atlas**. دي خطوات بسيطة ومجانية:

## 1. إنشاء الحساب
1. ادخل على موقع [MongoDB Atlas](https://www.mongodb.com/cloud/atlas/register).
2. سجل حساب جديد (ممكن تستخدم Google).

## 2. إنشاء قاعدة البيانات (Cluster)
1. بعد التسجيل، هيسألك شوية أسئلة، اختار أي حاجة أو تخطاها.
2. اختار الخطة المجانية **"M0 Free"**.
3. اضغط **"Create"**.

## 3. إعداد المستخدم (Database User)
1. هيطلب منك تعمل "Username" و "Password" لقاعدة البيانات.
2. اكتب اسم وباسورد (واحفظ الباسورد دي كويس عشان هنحتاجها).
3. اضغط **"Create User"**.

## 4. السماح بالاتصال (Network Access)
1. في القائمة الجانبية، اختار **"Network Access"**.
2. اضغط **"Add IP Address"**.
3. اختار **"Allow Access from Anywhere"** (أو اكتب `0.0.0.0/0`).
4. اضغط **"Confirm"**.

## 5. الحصول على الرابط (Connection String)
1. في القائمة الجانبية، ارجع لـ **"Database"**.
2. اضغط على زرار **"Connect"** جنب الـ Cluster بتاعك.
3. اختار **"Drivers"**.
4. تأكد إن الاختيار هو **"Node.js"**.
5. انسخ الرابط الطويل اللي هيظهرلك. شكله بيكون كده:
   `mongodb+srv://<username>:<password>@cluster0.abcd.mongodb.net/?retryWrites=true&w=majority`

## 6. تشغيل الموقع
1. ارجع للملفات عندك على الجهاز.
2. اعمل ملف جديد اسمه `.env.local` (جنب `package.json`).
3. اكتب جواه السطر ده، وبدل الكلام اللي بين قوسين بالبيانات بتاعتك:
   ```env
   MONGODB_URI=الرابط_اللي_نسخته
   ```
   *(ملاحظة: شيل `<password>` وحط الباسورد اللي عملتها في خطوة 3)*.

4. احفظ الملف، واعمل ريستارت للسيرفر (`Ctrl+C` وبعدين `npm run dev`).
